Overview

CiphereCore is a multi-purpose desktop security platform that combines military-grade encryption, secure messaging, and professional security tools into a single application. Built with Python and CustomTkinter, it offers a modern, elegant interface with enterprise-level security features.

Key Capabilities

  • Text Encryption — 5 algorithms (AES-256, Fernet, ChaCha20, Triple DES, AES-CTR)
  • File Encryption — Protect files up to 500MB with AES-256
  • Encrypted Chat — Server, Global, and Private messaging modes
  • Security Tools — Password generator, hash calculator, Base64, QR codes, UUIDs, and more
  • User System — Authentication with bcrypt, sessions, and friend management

Installation

CiphereCore is distributed as a standalone Windows executable. No Python installation or additional dependencies are required.

  1. Download CiphereCore.exe from the download page
  2. Place it in your preferred directory
  3. Double-click to launch

If Windows SmartScreen blocks the application, click "More info" → "Run anyway". This occurs because the executable is not yet code-signed.

First Launch

On first launch, CiphereCore will automatically:

  • Create a local SQLite database (ciphercore.db) for user accounts and messages
  • Create a logs/ directory for system logging
  • Create a certs/ directory for SSL certificates
  • Generate a config.json file for application settings

Authentication

CiphereCore uses a secure authentication system:

  • Password Hashing: bcrypt with 12 rounds of salt
  • Session Tokens: Cryptographically secure random tokens (32 bytes, URL-safe)
  • Rate Limiting: Maximum 5 login attempts before 5-minute lockout
  • Input Validation: Usernames 3-32 characters, alphanumeric with hyphens/underscores
# Registration example flow
Username: my_username    (3-32 chars, alphanumeric + - _)
Password: ••••••••       (any password accepted)
Email:    optional@email.com  (optional field)

Text Encryption

Navigate to the Encrypt tab to encrypt and decrypt text. Select your preferred algorithm, enter a password, and click Encrypt or Decrypt.

File Encryption

Navigate to the File tab to encrypt or decrypt files. CiphereCore supports all file types up to 500MB.

  • Select a file using the file browser
  • Enter an encryption password
  • Click Encrypt File — output is saved as filename.ext.enc
  • To decrypt, select the .enc file and enter the same password

Algorithms In-Depth

AES-256 (Recommended)

Advanced Encryption Standard with a 256-bit key in Cipher Block Chaining (CBC) mode. Uses PKCS7 padding and random 16-byte initialization vectors. This is the industry standard for government and military data protection.

Fernet

A symmetric encryption scheme that provides authenticated encryption using AES-128-CBC with HMAC-SHA256 verification. The key is derived from your password using SHA-256 and encoded as a URL-safe base64 string.

ChaCha20

A high-speed stream cipher designed by Daniel Bernstein. Uses a 256-bit key with a 96-bit nonce. Particularly efficient on devices without AES hardware acceleration.

Server Chat

Connect to a CiphereCore chat server to communicate with other users in real-time.

# Connect to a server
Host: 192.168.1.100    (server IP address)
Port: 5000             (default port)
Nickname: your_name
Password: server_pwd   (if server requires auth)

Global Chat

Global chat uses MongoDB for message persistence. All users connected to the same MongoDB instance can see global messages. Messages are stored in the global_chat collection.

Private Messages

Send direct messages to friends. Private messages are stored in the private_chat MongoDB collection and are only visible to the sender and recipient.

Hosting a Server

CiphereCore includes a built-in chat server. To host:

  1. Navigate to the Chat tab
  2. Click Server mode
  3. Set a port (default: 5000) and optional password
  4. Click Start Server
  5. Share your IP address and port with others to connect
# Server configuration
Port: 5000           (1024-65535)
Password: optional   (protects against unauthorized access)
Max connections: unlimited

Password Generator

Generate cryptographically secure random passwords using Python's secrets module. Customize length and character types (uppercase, lowercase, digits, special characters).

Hash Calculator

Compute hash values for any text input:

  • MD5 — 128-bit (not recommended for security)
  • SHA-1 — 160-bit (deprecated for security)
  • SHA-256 — 256-bit (recommended)
  • SHA-512 — 512-bit (maximum security)

All Tools

  • Password Generator — Secure random password creation
  • Hash Calculator — MD5, SHA-1, SHA-256, SHA-512
  • Base64 Encoder/Decoder — Encode and decode Base64
  • QR Code Generator — Create QR codes from text/URLs
  • UUID Generator — Generate UUID v4 identifiers
  • Color Picker — Pick colors with hex/RGB output
  • Text Analyzer — Character, word, and line counting
  • File Checksum — Verify file integrity with MD5/SHA

Configuration File

CiphereCore stores settings in config.json located in the application directory:

{
  "theme": "dark",
  "default_port": 5000,
  "default_cipher": "AES-256",
  "auto_connect": false,
  "message_encryption": true
}

MongoDB Setup (Optional)

For global and private chat persistence, CiphereCore connects to MongoDB. Configure via environment variables or .env file:

# .env file
MONGO_URI=mongodb://localhost:27017/
MONGO_DB_NAME=ciphercore

MongoDB is optional — the application works fully without it using local SQLite storage. MongoDB enables global chat, private messaging, and friend system features.

Required Collections (auto-created)

  • users — User accounts with indexed username
  • social — Friends and friend requests
  • global_chat — Global broadcast messages
  • private_chat — Direct messages between users
  • server_chat — Per-server channel messages
  • logs — Server-side event and error logs
